# OpenNeuro Utilities

This package provides utilities for building pipelines that process publicly available datasets from [OpenNeuro](https://openneuro.org/).

**📖 Documentation can be found in the official brainsets docs:** See [Creating an OpenNeuro Pipeline](https://brainsets.readthedocs.io/en/latest/concepts/openneuro_pipeline.html) for the complete guide on building OpenNeuro pipelines.

## API Reference

For detailed API documentation, see [`brainsets.utils.openneuro`](https://brainsets.readthedocs.io/en/latest/package/utils.html#openneuro).
